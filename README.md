# Description
Package for updating ionic, angular-capacitor projects

# Install
`npm install  -g vesup`

# Config
## Android
```
{
    "files":[
        {
            "platform": "android",
            "path": "android/app/build.gradle"
        }
    ]
}
```

# Use
```
vesup major
```
```
vesup minor
```
```
vesup patch
```
